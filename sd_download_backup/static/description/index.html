Copias de seguridad

Descripción

Resúmen

Módulo para enlazar el directorio con las copias de seguridad realizadas por sistemas de datos con la gestión de Odoo. Con un enlace al FTP se accede a las copias de seguridad realizadas por el script: backup-all.sh realizado por Sistemas de Datos.

Configuración previa

Configurar servidor FTP y directorios

Debemos saber en qué servidor se está ejecutando la instancia Odoo a configurar, ya que los backups se almacenan en directorios distintos.

Nos dirigimos al menú: "Configuración/Técnico/Parámetros/Backup FTP"

Creamos una configuración donde establecemos lo siguiente:

  • Servidor: servidor FTP (dependiendo de la máquina donde se esté ejecutando la instancia de Odoo)
  • Directorio en el servidor FTP: ruta física donde se almacenan las copias en el servidor
  • Usuario: usuario UNIX de la máquina para abrir el FTP
  • Contraseña: contraseña del usuario UNIX

Establecida la configuración ejecutamos el botón inicializar.

Permiso a usuario

En la configuración del usuario debemos activar el permiso "Descargar backup" para tener acceso al menú.

Uso

Descargar copias de seguridad

Nos dirigimos al menú "Copias de seguridad/Descargar los Backups".

Como podemos ver hay almacenada una semana de copias ordenada por última creación, todos los días se ejecuta una tarea a las 4:00AM para actualizar la información de los backups.

Para descargarla simplemente presionamos la flechita verde de la derecha, dependiendo de la copia que querramos descargar.
