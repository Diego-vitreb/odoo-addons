Copias de seguridad

Descripción

Resúmen

Módulo para enlazar el directorio con las copias de seguridad realizadas por sistemas de datos con la gestión de Odoo. Mediante enlaces simbólicos se accede a las copias de seguridad realizadas por el script: backup-all.sh realizado por Sistemas de Datos.

Configuración previa

Crear directorio y enlaces

Nos situamos en el directorio (físico) de almacenamiento de adjuntos, por defecto es "~/.local/share/Odoo/filestore/nom_bbdd/". Debemos crear un directorio nuevo con el nombre "backups".

Dentro del directorio debemos crear los enlaces simbólicos. Los nombres deben ser los días de la semana exceptuando el Domingo.

Permiso a usuario

En la configuración del usuario debemos activar el permiso "Descargar backup" para tener acceso al menú.

Por seguridad, a pesar de tener los botones visibles, no podremos editar ni eliminar los backups. Este derecho es reservado solo para el usuario 1 (administrador)
