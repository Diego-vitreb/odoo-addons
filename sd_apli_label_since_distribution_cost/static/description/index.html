Etiquetas APLI en distribución de costo

Descripción

Resúmen

Módulo desarrollado para generar etiquetas APLI desde una distribución de gastos. Con este módulo podemos generar un fichero txt o un xls con las etiquetas para cargarlo desde la aplicación de apli.

Configuración

Tipo de archivo generado

Previamente debemos configurar el tipo de archivo que vamos a generar (TXT o XLS).

En el directorio del módulo, dentro de wizard, debemos editar el fichero __init__.py en el cual le decimos que fichero .py debe cargar. Solo se puede cargar uno.

APLI

En el directorio "doc" dentro del módulo encontraremos el programa APLI y una configuración de etiquetas, esa será la que debemos editar y ponerle al cliente.

En el cliente deberemos generar un fichero de etiquetas y dentro del programa de APLI cargarlo como base de datos. Cada vez que generemos un fichero nuevo, debemos sobrescribir el anterior.

Funcionamiento

Generar fichero

Una vez la distribución se encuentre en estado realizado, nos dirigimos al botón "Más", "Etiquetas de productos".

Se abrirá un wizard donde seleccionará todos los productos con las cantidades que hay en la distribución, por lo que se generará una etiqueta para cada unidad de producto.

En caso de que queramos imprimir más cantidades (o menos), siempre podemos editar el número de cantidad para imprimir las etiquetas que deseamos. También podemos eliminar artículos, por los teníamos etiquetados previamente.

Una vez conformes con el número de etiquetas que vamos a imprimir, pulsamos el botón "Generar fichero". Con ello se ejecutará el código de generación del archivo para posteriormete poder descargarlo.

También podemos generar etiquetas desde el menú "Ventas/Productos/Variantes de producto".

Al marcar varios, en el botón "Más" encontraremos "Etiquetas de productos".

Fichero generado

En el momento de darle a "Generar fichero" se abrirá otra vista para descargarlo.

En la imagen podemos apreciar la estructura de un fichero xls, en caso de ser un txt, la estructura será la misma pero separado por ";".
